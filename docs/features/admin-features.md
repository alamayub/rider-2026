# Admin Features

## Operations Dashboard
- Live trips monitoring across cities
- Active driver status board (online/offline/busy)
- Real-time ride state transitions and event stream
- Map view for active fleet movement

## City Configuration & Controls
- City creation and update controls
- Fare configuration (`baseFare`, `perKm`, taxes)
- City-level support contact management
- Service availability windows and rollout toggles

## User & Driver Management
- Rider and driver account lookup
- Driver verification queue management
- Account suspension/reactivation flows
- Manual intervention on problematic rides

## Financial & Risk Controls
- Payment webhook review and reconciliation status
- Refund/dispute lifecycle handling
- Fraud signal visibility and action logging
- Promo/coupon configuration governance

## Reporting & Compliance
- Ride completion/cancellation trend metrics
- City-level KPI snapshots (ETA, acceptance, failures)
- Admin action audit logs
- Compliance-ready data retention and event history access
